# flightschedule

A flight schedule application built with Next.js and TypeScript.

## Features

- Flight schedule display
- Airport information
- Responsive design with Tailwind CSS
- Ad integration support

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
